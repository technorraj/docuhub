# DocumentaryHub

DocumentaryHub is a Netflix/Prime Video-inspired documentary streaming platform built with PHP, MySQL, Bootstrap, JavaScript, and HTML/CSS. It aggregates free documentaries from YouTube and other free sources, providing a clean and modern viewing experience without hosting videos directly.

## Features

* 🎬 Documentary-only platform
* 🔍 Search and category filtering
* ⭐ Featured and trending documentaries
* ❤️ Favorites and Watch Later
* 📜 Watch history and Continue Watching
* 👤 User authentication system
* 🛠️ Admin panel for content management
* 📱 Responsive Prime Video-inspired UI
* 🔄 Automatic content updates using YouTube Data API and cron jobs
* 🔒 Secure PHP code using PDO and prepared statements

## Tech Stack

* PHP
* MySQL
* HTML5
* CSS3
* Bootstrap 5
* JavaScript

## Data Sources

Documentaries are embedded from free sources such as:

* DW Documentary
* Free Documentary
* BBC Earth
* National Geographic
* PBS

DocumentaryHub does not host any videos. All videos remain the property of their respective creators and are displayed using embedded players.

This project is intended for educational purposes and as a beginner-friendly full-stack PHP project.
